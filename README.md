# IIT Kharagpur ERP Modernized UI

A modernized, responsive, light-themed premium dashboard for the IIT Kharagpur ERP System. Designed with clean semantics, elegant glassmorphism accents, and a brand-focused color palette (`#291973`).

## Features
- **Modern Aesthetic**: Glassmorphism cards with smooth hover transitions, ambient color blobs, and outfit typography.
- **Premium Light Theme**: High-contrast, sleek light background utilizing `#291973` as the primary brand identity.
- **Fully Responsive**: Collapsible sidebar, mobile navigation, and optimized grids for all screen sizes.
- **Complete Semantic Structure**: Single `h1` headings, distinct `nav`, `aside`, `main`, and `footer` elements.
- **Working Prototype Components**:
  - Interactive profile dropdown menu.
  - Interactive sidebar accordion (e.g. Academic module).
  - Sliding drawer navigation for mobile screens.

## Structure
- `/index.html` - Home Dashboard view containing overview statistics, ERP e-Office links, and quick actions.
- `/style.css` - Unified design tokens, global utility classes, animations, and component styles.
- `/academic/index.html` - Integrated Academic Module submenu demonstrating examination portals and timetable listings.
